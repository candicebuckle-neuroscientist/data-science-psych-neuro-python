This repository contains Python projects and exercises completed as part of the **INCF Data Science for Psychology and Neuroscience** course.

## Course Overview
The course focuses on applying Python for data analysis in psychology and neuroscience, covering:
- Data manipulation with **Pandas**
- Numerical computing with **NumPy**
- Data visualization with **Matplotlib** and **Seaborn**
- Basic statistical analysis and plotting

**Course link:** [INCF Data Science Course](https://training.incf.org/lesson/data-science-psychology-and-neuroscience-python)

## Skills Demonstrated
- Writing clean, well-documented Python code
- Loading, cleaning, and exploring datasets
- Creating visualizations for data interpretation
- Performing basic statistical analysis
- Organizing projects and reproducible workflows
